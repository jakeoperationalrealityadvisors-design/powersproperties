# Asset Management for Powers Construction & Properties

## Folder Structure
- `/branding/` - Company logos and branding materials
- `/rentals/` - Images for rental properties
- `/projects/` - Before/after images for construction projects
- `/ui/` - User interface elements and backgrounds

## Naming Conventions
- All filenames must be lowercase
- Use hyphens (-) to separate words, no spaces
- Image formats: .jpg for photos, .png for logos/graphics
- Rental images: unit[number]-[image_number].jpg (e.g., unit1-1.jpg)
- Project images: project[number]-[before|after].jpg (e.g., project1-before.jpg)
- Branding: logo.png
- UI: descriptive names like hero-bg.jpg, placeholder.jpg

## Optimization
- Keep images lightweight and web-optimized
- Use appropriate resolutions for web display
